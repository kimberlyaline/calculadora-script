# Calculadora em Python e Shell Script

Este repositório apresenta uma calculadora que funciona diretamente pelo terminal, criada em duas implementações diferentes:

*Uma versão em Python (Aula1.pynb)

*Uma versão em Shell Script (calculadora.sh)

O projeto foi elaborado como parte de uma atividade prática do curso de Análise de Dados da EBAC, utilizando controle de versões com o GitHub Desktop.
